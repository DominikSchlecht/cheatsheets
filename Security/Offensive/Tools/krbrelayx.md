## dnstool.py
#dnstool
Create new entry
`python3 dnstool.py -u <DOMAIN>\\<User> -p <password> -a add -r <target-dns-name> -d <dns-target-ip> <target>`

Check with query
`python3 dnstool.py -u <DOMAIN>\\<User> -p <password> -a query -r <target-dns-name> <target>`
