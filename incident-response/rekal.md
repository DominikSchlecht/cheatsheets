# Basic Usage

## Read from File:
`rekal[.exe] -f <file>`

## Live Analysis
Windows:
`rekal.exe live`

Linux:
`rekal --live`

## List plugins
In Rekal:
`plugins.<tab><tab>`

# Advanced Stuff
## Find Prog owning specific memory adress / find virtual addr from physical addr
In Rekal: `ptov(<adress>)`

Where `ptov` stands for physical to virtual
